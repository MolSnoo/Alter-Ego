# Contributing to Alter Ego

Thank you for your interest in contributing to Alter Ego! The following are some
ways that you can contribute to Alter Ego.

## Bug Reports

Please submit bug reports on our [GitHub Issue Tracker](https://github.com/MolSnoo/Alter-Ego/issues).

Before submitting a bug report please ensure that you can reproduce the bug
and you provide logs and screenshots in the issue. This helps us diagnose and
fix the bug faster.

## Pull Requests

If you have an feature or fix that you want to get merged, then please submit a
pull request on [GitHub](https://github.com/MolSnoo/Alter-Ego/pulls).

Before submitting a pull request for review, please ensure that all tests in the
test suite has passed.

## Getting Help

If you need any help in making a contribution for Alter Ego, please make an issue
on our [GitHub Issue Tracker](https://github.com/MolSnoo/Alter-Ego/issues). We
would be more than happy in guiding you through the process.
